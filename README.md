# Mr Robot VAPT Report

## Project Overview

This repository contains an Internal Network Penetration Testing (VAPT) report for the Mr. Robot Vulnerable Machine. The assessment was performed in a controlled lab environment for educational and ethical cybersecurity learning.

## Scope

- Target: Mr Robot Vulnerable Machine
- Environment: Internal Lab
- Operating System: Kali Linux
- Assessment Type: Internal VAPT

## Methodology

- Information Gathering
- Network Scanning
- Vulnerability Assessment
- Exploitation
- Post Exploitation
- Privilege Escalation
- Reporting

## Tools Used

- Nmap
- Netdiscover
- Burp Suite
- WPScan
- Metasploit Framework
- Linux Commands

## Vulnerabilities Identified

- WordPress Information Disclosure
- Weak Login Credentials
- Privilege Escalation via SUID Binary

## Skills Demonstrated

- Network Enumeration
- Web Application Security
- Vulnerability Assessment
- Exploitation
- Privilege Escalation
- VAPT Reporting
